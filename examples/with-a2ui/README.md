# with-a2ui

A2UI declarative UI rendering alongside AG-UI chat.

## Quick Start

```bash
npx assistant-ui@latest create my-app --example with-a2ui
```

## Manual Setup

1. Install dependencies: `pnpm install`
2. Run dev server: `pnpm dev`
3. Open http://localhost:3000

## Features

- A2UI declarative surface rendering alongside AG-UI chat
- Reactive data binding with JSON pointer paths
- User actions flowing back to the agent
- Custom styled component overrides
- Mock booking agent (no API key required)
